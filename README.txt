# نظام الاستعلام الذكي للناخبين - الخليل

ملفات الموقع:
- index.html
- styles.css
- app.js
- data.js

طريقة التشغيل:
1. افتح الملف `index.html` مباشرة في المتصفح، أو ارفعه إلى GitHub Pages.
2. جميع البيانات مضمنة محليًا داخل `data.js`.
3. لا توجد قاعدة بيانات أو خادم خلفي.

ملاحظة:
- الملف الأصلي لا يحتوي على عمود مستقل لرمز مركز الاقتراع، لذلك تم توليد رمز داخلي لكل مركز بصيغة H-001 / H-002 ...
